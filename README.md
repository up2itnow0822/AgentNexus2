## ðŸš€ Quick Start

### Prerequisites
- Node.js 18+
- Docker
- PostgreSQL
- Foundry (for smart contracts)

### Installation

```bash
# Clone repository
git clone https://github.com/yourusername/AgentNexus2.git
cd AgentNexus2

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env
# Edit .env with your API keys

# Set up database
cd backend
pnpm prisma migrate dev
pnpm prisma db seed

# Start development servers
cd ..
pnpm dev
```
