# Project Status

AgentNexus is live on Base mainnet as of December 2025.

## Current State

**Version**: v0.1.0 (Base Mainnet Launch)

The project is:
- **Actively maintained**
- **Early-stage**
- **Infrastructure-focused**

## Not Production-Hardened For

The project is NOT production-hardened for:
- Retail users
- Untrusted agents
- Large-value transactions
- Regulatory compliance (toggles available but disabled by default)

## Stability

Interfaces may change without notice.

Breaking changes will be documented in release notes when possible, but backwards compatibility is not guaranteed during this phase.
